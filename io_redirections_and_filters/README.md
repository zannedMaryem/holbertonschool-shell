#Shell, I/O Redirections and filters
## Description : This project contains shell scripts that demonstrate the use of input/output redirections, filters.
Each script is executable and tested in a sandbox environment
 ## Scripts
 - **0-hello_world**
 Prints "Hello, World" followed by a new line.
 Usage: `./0-hello_world`

 - **1-confused_smiley**
 Displays a confused smiley `"(Ôo)'`.
 Usage: `./1-confused_smiley`

 - **2-hellofile**
 Displays the third line of the file `iacta`.
 Usage: `./2-hellofile`

 - **3-twofiles**
 Creates a file named `*\'"Best School"\'*$?*****:)` containing the text `Best School`.
 Usage: `./3-twofiles`
 - **4-lastlines**
 Writes the result of `ls -la` into `ls_cwd_content`.
 Usage: `./4-lastlines`
 - **5-firstlines**
 Duplicates the last line of the file `iacta`.
 Usage: `./5-firstlines`
 - **6-third_line**
 Deletes all `.js` files in the current directory and subfolders.
 Usage: `./6-third_line`
 - **7-file**
 Counts the number of directories and subdirectories (excluding `.` and `..`).
 Usage: `./7-file`
 - **8-cwd_state**
 Displays the 10 newest files in the current directory.
 Usage: `./8-cwd_state`

 ## Requirements
 - Scripts must be exactly two lines long (shebang + command).
 - All scripts are executable (`chmod u+x filename`).
 - Tested on Ubuntu 20.04 LTS.
